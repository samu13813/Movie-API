Business Logic URL HTTP Method Request body data format Response body data format
Gets a list of all movies /movies GET None A JSON object holding data about all the movies by name
Gets the data of the director by name /movies/director/[NAME] GET None A JSON object holding data about the director, containing name and date of birth. Example:
{
  name: "Christopher Nolan",
  dob: "30/07/1970"
}
Allows new user to register /users POST A JSON object holding data about the user to add, structured like:
{
  name: "Sam",
  favoriteMovies: [ ]
}
A JSON object holding data about the new user added, including a unique ID. Example:
{
  name: "Sam",
  id: "7cb380cb-a1bc-48f7-ab06-1e69c2f88bb4",
  favoriteMovies: [ ]
}
Allows the user to update their username /users/[NAME]/[NEWNAME] PUT None A text message idicating whether the username was succesfully updated.
Allows user to add a movie to their list of favorites /users/[NAME]/[MOVIETITLE] POST None A text message indicating whether the movie was succesfully added
Allows user to remove a movie from their list of favorites /users/[NAME]/[MOVETITLE] DELETE None A text message indicating whether the movie was succesfully removed
Allows user to delete their username /users/[NAME] DELETE None A text indicating whether the username was succesfully deleted
